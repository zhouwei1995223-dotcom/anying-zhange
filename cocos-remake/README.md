# Anying Cocos Remake

Cocos Creator 3.8.8 2D project skeleton.

Open this folder with:

```text
C:\ProgramData\cocos\editors\Creator\3.8.8\CocosCreator.exe
```

First milestone: import processed PNG assets, create one scene, and verify it can preview in browser.

Viewport target:

- Design size: `1600x900`
- Aspect ratio: `16:9`
- First web target: desktop browser and large laptop screens

Imported starter assets:

- `assets/resources/maps/sm_0.jpg`
- `assets/resources/heroes/hero_00/hero_00_sheet.png`
- `assets/Main.scene`

To preview: open `assets/Main.scene`, then run Preview.
