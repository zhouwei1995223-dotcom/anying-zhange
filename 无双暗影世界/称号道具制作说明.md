# 称号道具制作说明

本说明用于后续新增称号道具。当前称号系统使用独立装备位 `title`，不会占用耳环、戒指、项链或时装格。

## 当前已做好的称号

- 道具编号：`1713049900`
- 道具名称：`天下无双`
- 道具类型：饰品类道具，`goodsType=3`
- 称号装备类型：`dType=4`
- 光效编号：`goodsLuck=18`
- 角色存档字段：`<title>1713049900</title>`

## 配置文件位置

称号道具配置在：

`D:\无双暗影世界\无双暗影世界\server\config\data\jewels\jewels.txt`

这个文件原本是饰品配置。称号目前也走饰品数据结构，但 `dType` 单独使用 `4`，代表称号。

## 关键字段

以 `1713049900` 这一行为例，重点看这些列：

| 作用 | 当前值 | 说明 |
| --- | --- | --- |
| 道具编号 | `1713049900` | GM 指令加物品时用这个编号 |
| 名称 | `天下无双` | 鼠标悬浮显示的道具名 |
| goodsType | `3` | 饰品类道具 |
| dType | `4` | 称号专用装备位 |
| lifeRaised | `500` | 增加生命上限 |
| manaRaised | `300` | 增加魔法上限 |
| physicalDefense | `20` | 物理防御 |
| magicDefense | `20` | 魔法防御 |
| hittingAbility | `15` | 命中 |
| evadingAbility | `15` | 闪避 |
| goodsLuck | `18` | 头顶/角色光影特效编号 |

后续新增称号时，建议复制 `1713049900` 这一行，改成新的编号、名称、属性和 `goodsLuck`。

## 图片和特效

称号显示不是普通装备图标决定的，而是通过 `goodsLuck` 指向客户端光效资源。

当前 `天下无双` 使用：

- `goodsLuck=18`
- 对应你之前做好的称号光效图片/IMG 资源

新增称号时：

1. 先做好 PNG。
2. 用 PNG/IMG 转换工具转成游戏可读的 `.img`。
3. 放到客户端对应光效资源位置。
4. 在 `jewels.txt` 新称号行里把 `goodsLuck` 改成对应特效编号。

## 角色存档装备称号

角色存档位置示例：

`D:\无双暗影世界\无双暗影世界\server\data\depot\dddd.txt`

装备称号字段：

```xml
<title>1713049900</title>
```

没有装备称号时：

```xml
<title></title>
```

## GM 添加道具

如果要给角色发称号道具，用称号编号：

```text
1713049900
```

后续新增称号，就使用新称号那一行的道具编号。

## 注意事项

- 称号必须是 `goodsType=3` 的饰品类数据。
- 称号专用装备位必须设置 `dType=4`。
- `goodsLuck` 控制头顶特效编号。
- 修改 `jewels.txt` 后必须重启服务端。
- 修改客户端 `.img` 或 `client.zip` 后必须重启客户端。
- 如果称号能穿戴但没有特效，优先检查 `goodsLuck` 和客户端光效资源是否存在。
- 如果称号不能添加出来，优先检查道具编号是否在 `jewels.txt` 中存在，且服务端是否已经重启。
