# 称号和光环新增说明

## 配置文件

称号物品放这里：

```txt
server/config/data/jewels/titles.txt
```

光环物品放这里：

```txt
server/config/data/jewels/halos.txt
```

普通首饰继续放这里：

```txt
server/config/data/jewels/jewels.txt
```

## 素材编号规则

称号看这一列：

```txt
称号素材编号
```

光环看这一列：

```txt
光环素材编号
```

这两列下面的英文参数名都是：

```txt
goodsLuck
```

不要改英文参数名，游戏靠它读取素材编号。

现在已经改成纯数字读取：

```txt
素材编号=100  读取 client/pak/effect.zip 里的 image/effect/100.img
素材编号=101  读取 client/pak/effect.zip 里的 image/effect/101.img
素材编号=102  读取 client/pak/effect.zip 里的 image/effect/102.img
```

当前对应关系：

```txt
天下无双    称号素材编号=100    读取 image/effect/100.img
恭喜发财    光环素材编号=101    读取 image/effect/101.img
```

## 新增素材放哪里

新做好的称号或光环素材，要放进客户端资源包：

```txt
client/pak/effect.zip
```

在压缩包里的路径必须是：

```txt
image/effect/数字.img
```

例如你新做一个称号素材，想编号 `102`：

```txt
client/pak/effect.zip/image/effect/102.img
```

然后在 `titles.txt` 里把这个称号的“称号素材编号”填成 `102`。

## 新增称号流程

1. 在 `server/config/data/jewels/titles.txt` 复制一行旧称号。
2. 改新道具编号、名字、说明、价格、属性。
3. 把“称号素材编号”改成新素材编号，比如 `102`。
4. 把素材文件放进 `client/pak/effect.zip/image/effect/102.img`。
5. 在 `client/data/titles.txt` 里加一行：`道具编号 素材编号 名字`。
6. 重启服务端和客户端后测试。

## 新增光环流程

1. 在 `server/config/data/jewels/halos.txt` 复制一行旧光环。
2. 改新道具编号、名字、说明、价格、属性。
3. 把“光环素材编号”改成新素材编号，比如 `103`。
4. 把素材文件放进 `client/pak/effect.zip/image/effect/103.img`。
5. 在 `client/data/halos.txt` 里加一行：`道具编号 素材编号 名字`。
6. 重启服务端和客户端后测试。

## 注意

`title.img` 和 `aura_dragon.img` 是旧命名素材，已经不作为称号/光环的主读取方式。

以后新增时优先使用数字素材，例如 `102.img`、`103.img`、`104.img`。
